# Airsoft Arena Live Player Count - Fixed PIN Version

Use this version if you want the PIN system.

## First-time setup
1. Install Node.js
2. Open this folder in Command Prompt
3. Run:
npm install

## Start the site
Windows Command Prompt:
set ADMIN_PIN=1234 && npm start

## Open the site
http://localhost:3000

## Notes
- The server trims spaces from the PIN
- The page trims spaces from the PIN
- The page shows the PIN length loaded by the server
